# atopile-dev Agent Skill

> **This skill was generated by an AI agent.** Contributions, corrections, and improvements are welcome!

An [agentskills.io](https://agentskills.io)-compatible skill that teaches AI agents how to develop with [atopile](https://atopile.io)'s declarative DSL for PCB and electronics design.

## Install

### Claude Code

Add to your `.claude/settings.json`:

```json
{
  "agentSkills": [
    {
      "path": "agent-skill"
    }
  ]
}
```

Or install from GitHub:

```json
{
  "agentSkills": [
    {
      "uri": "https://github.com/atopile/atopile/tree/main/agent-skill"
    }
  ]
}
```

### Cursor / Windsurf

Copy `SKILL.md` and the `references/` directory into your project's rules directory (e.g. `.cursor/rules/` or `.windsurf/rules/`).

### Manual

Copy the `agent-skill/` directory into your project and point your agent tool at `SKILL.md`.
